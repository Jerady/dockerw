# ubuntu docker image

## jerady/ubuntu

Ubuntu Docker Image based on ubuntu:16.04
contains:
* byobu
* curl
* git
* htop
* locate
* man
* unzip
* vim
* wget
* mc

AND

Oracle Java 8 JRE 8u74

## build
./dockerw build

## Run (standalone)
./dockerw run

## Stop
./dockerw stop

## Clean up
./dockerw clean
